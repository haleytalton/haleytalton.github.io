This module was written in Python 2.7 using the PyCharm IDE.

Run the program as a module by:
1) locating the path of your input file
2) entering 'python -m proj1 input_file_location output_file_location'
    into the command line

The input file should be a txt file containing prefix expressions.
The output file will echo the input expressions and provide the corresponding
postfix form of the input.

The file containing the output will be written in the specified location
you provide for the output file.

Only prefix expressions with letters and operators are accepted.
Expressions with spaces are accepted, but the output expression will
have no spaces.
